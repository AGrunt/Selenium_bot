# Selenium_bot

## Installation

1. Create virtual environment. Run `python -m venv .venv`
2. Activate virtual environment. Run `source .venv/bin/activate`
3. Install requirements. Run `pip install -r requirements.txt `

## How to run

1. Run `python bot.py`
